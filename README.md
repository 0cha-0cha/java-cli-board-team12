# 프로그래머스 데브코스

## 2차 12팀 [말랑말랑] - Java CLI Board

콘솔 환경에서 동작하는 게시판 프로젝트입니다.
메뉴 기반으로 게시글 등록, 목록 조회, 상세 조회, 수정, 삭제 기능을 제공합니다.

---

| 이름 | GitHub ID | 주요 기여 |
|---|---|---|
| 손주현 | @0cha-0cha | 메인 메뉴 흐름 연결, 게시글 삭제 기능 구현 |

---

## 기능 한눈에 보기

| 기능 | 상태 | 핵심 내용 |
|---|---|---|
| 게시글 등록 | 완료 | 작성자/제목/내용 입력, 빈 값 방지 |
| 전체 게시글 조회 | 완료 | 고정폭 표 형식 목록 출력 |
| 게시글 상세 조회 | 완료 | ID 조회 + 조회수 증가 + 본문 출력 |
| 게시글 수정 | 완료 | 기존 글 확인 후 제목/내용 수정 |
| 게시글 삭제 | 완료 | ID 기반 삭제, 성공/실패 안내 |

> 프로그램 시작 시 샘플 게시글 3개가 자동 생성됩니다.

---

## 사용 가능한 명령어

| 메뉴 번호 | 명령 | 설명 |
|---|---|---|
| 1 | 게시글 등록 | 새 게시글 작성 |
| 2 | 전체 게시글 조회 | 게시글 목록 출력 |
| 3 | 게시글 상세 조회 | 게시글 1건 상세 출력 |
| 4 | 게시글 수정 | 기존 게시글 제목/내용 수정 |
| 5 | 게시글 삭제 | 게시글 삭제 |
| 0 | 종료 | 프로그램 종료 |

---

## 실행 방법

### 1) 컴파일

```bash
javac -d out src/Main.java src/data/*.java src/function/*.java src/main/*.java
```

### 2) 실행

```bash
java -cp out Main
```

│  └─ main/

│     ├─ ProgressBar.java       # 로딩 표시
│     └─ ClearScreen.java       # 콘솔 화면 정리
